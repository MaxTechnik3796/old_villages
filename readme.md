The mod adds old villages from version 1.13 - as new structures.

Villages spawn in plains, deserts, savannas and taiga biomes.

Some houses have workstations for villagers or beds.

I recommend using the Programmer Art Resource Pack.

The blacksmith obviously has his own loot chest.

The mod includes standard settings and several additional features, such as 1.4-broken generation of torches, changes between Gravel and Dirt Path, and the disabling of vanilla villages.
